# Schnappix Menü Setup

## Menü-Struktur für Hamburger-Menü

Erstelle in Shopify Admin (Online Store → Navigation) ein Menü mit dem Namen **"schnappix"** mit folgenden Links:

### Hauptmenü-Links für Schnappix:

1. **Home**
   - URL: `/` oder Homepage
   - Typ: Homepage Link

2. **Produkte**
   - URL: `/collections/all` oder Haupt-Kollektion
   - Typ: Collection Link

3. **Über Uns**
   - URL: `/pages/about-us` oder `#about-us`
   - Typ: Page Link oder Custom Link

4. **Standorte**
   - URL: `/pages/standorte` oder `#shops`
   - Typ: Page Link oder Custom Link

5. **Support**
   - URL: `/pages/support` oder `#support`
   - Typ: Page Link oder Custom Link

6. **Kontakt**
   - URL: `/pages/kontakt` oder `#contact`
   - Typ: Page Link oder Custom Link

### Farb-Schema (Scheme 6) - Schnappix Brand Colors

Das Menü verwendet automatisch die Schnappix-Farben aus **Scheme 6**:

- **Hintergrund**: #FFFFFF
- **Headings**: #000000
- **Text**: #170B02
- **Links**: #5D2B09
- **Hover Links**: #170B02
- **Borders**: #F7F1ED
- **Primary Button Background**: #5D2B09
- **Primary Button Text**: #F7F1ED
- **Primary Button Hover**: #98591B
- **Secondary Button**: Transparent mit #170B02 Text/Border
- **Inputs**: #FFFFFF Background, #5D2B09 Border

### Menü-Position

Das Menü ist **rechts im Header** positioniert und wird als **Hamburger-Menü** angezeigt (sowohl Desktop als auch Mobile).

### Menü-Style

- **Menu Style**: Text (einfaches Text-Menü)
- **Color Scheme**: scheme-6 (Schnappix Brand Colors)
- **Accordion**: Deaktiviert (flaches Menü)

### Setup-Schritte:

1. Gehe zu: **Shopify Admin → Online Store → Navigation**
2. Klicke auf **"Add menu"** oder bearbeite das bestehende Menü "schnappix"
3. Füge die oben genannten Links hinzu
4. Speichere das Menü
5. Das Hamburger-Menü sollte automatisch die Links anzeigen

### Alternative: Menü über Theme Editor

Du kannst das Menü auch direkt im Theme Editor bearbeiten:
1. Gehe zu: **Shopify Admin → Online Store → Themes → Customize**
2. Klicke auf **Header**
3. Wähle das **Menu Block**
4. Wähle das Menü "schnappix" aus dem Dropdown
5. Die Links werden automatisch geladen
6. **Color Scheme** ist bereits auf "scheme-6" gesetzt
